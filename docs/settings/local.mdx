## Install `luminox-cli`
```bash
curl -fsSL https://install.luminox.io | sh
```

## Start Luminox Server Locally
Luminox requires at least an OpenAI API Key to Start.
```bash .env
LLM_API_KEY="YOUR_OPENAI_API_KEY"
```
Place your `.env` file and run `luminox docker up` at the same directory:
```bash server
mkdir luminox_server && cd luminox_server
luminox docker up
```
`luminox docker up` will create/use the existing `.env` and `config.yaml` for Luminox, and create a `db` folder to persist data.

Once it's done, you can access the following endpoints:
- Luminox Base URL: http://localhost:8029/api/v1
- Luminox Dashboard: http://localhost:3000/

<Info>
To use different providers, check [Core Settings](/settings/core) for more details.
</Info>
