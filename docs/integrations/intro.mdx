Luminox Community provides a collection of integrations for popular AI frameworks and tools.


## The easiest way to get started
Download `luminox-cli` and create a new project with the template:
```bash
curl -fsSL https://install.luminox.io | sh
luminox create my-proj --template-path "python/openai-agent-artifacts"
```


For more e2e integrations, please refer to the [Luminox-Examples](https://github.com/memodb-io/Luminox-Examples) repository.
